Comentarios iniciales

Este es el primer plugin que creo, y su finalidad ayudarme a crear mi juego deckbulider para navegador.

para considerar este proyecto un exito voy a tener que cumplir con una serie de expecativas:

  • el plugin debe ser compatible con telefonos celulares.
  • debe poder distingir entre un arrastre y un click.
  • debe ser lo suficientemente flexible para permitirme tener varios tipos de dropzones.
  • debe poder arrastrar divs complejos y no solo imagenes.

no estoy seguro de poder cumplir con todas estas espectativas, pero voy a hacer lo mejor que pueda

atributo draggable

el atributo draggable me permite arrastrar items, asi que tengo que asignarle draggable="true" a cada uno de los items

atributo por plugin

antes que otra cosa hay que llamar a jquery 

 <script src="https://code.jquery.com/jquery-1.12.4.min.js"></script>

ahora, para comenzar a crear este plugin voy a hacer que el atributo draggable sea asignado como true a todos los elementos con la clase item ( la cual le di a #item 1 2 y 3) 

// julif-dragdrop v 0.1
(function ( $ ) {
  
  $.fn.julif_dragdrop = function() {
    this.attr("draggable", "true")
    return this;
  };
}( jQuery ));
$( ".item" ).julif_dragdrop();
